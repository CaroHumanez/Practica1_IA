#  Sistema Inteligente de Recomendación - IA

Este proyecto integra un sistema experto, lógica difusa y una ontología para recomendar películas y series personalizadas según los gustos e intereses del usuario.

##  ¿Qué incluye?

-  **Sistema Experto** (Experta): 15 reglas con prioridades y control de ejecución.
-  **Lógica Difusa** (Scikit-Fuzzy): interés, apertura y edad → recomendabilidad.
-  **Ontología** (RDFLib): representación de contenido, géneros, formatos e idiomas.
-  **Interfaz Streamlit**: para interactuar con el sistema de forma visual.
